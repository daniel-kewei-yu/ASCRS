# Aperture Science Cursor Relocation Suite (ASCRS)

A fun utility that places two portals (blue and orange) on your screen.  
In **edit mode** you can drag the portals around.  
In **play mode** the cursor teleports from one portal to the other, preserving its relative position.

### Controls

- **Esc** – Exit the application.

### Features

- Custom application icon and edit mode cursor.
- Instant teleportation using a low-level mouse hook.
- Click‑through overlay in play mode, so you can interact with other windows.

### Requirements

- Windows 10/11
- .NET 8.0 Runtime

### Build

```bash
dotnet build
dotnet run
```
